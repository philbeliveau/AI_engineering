# Get Methodologies

Query the knowledge base for step-by-step methodologies and processes.

**Note:** This is a Registered tier endpoint - requires API key.

Use the knowledge-pipeline MCP server's `get_methodologies` tool to find methodologies about: $ARGUMENTS

If no arguments provided, list all available methodologies.

Present each methodology with:
- Name and purpose
- Prerequisites
- Step-by-step process
- Expected outputs
- Source attribution

Use these methodologies to inform BMAD workflow creation.
